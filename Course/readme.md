# 🌱 Estructura de la Carpeta `course`

## 📄 Archivos  
- **`app.js`**: Archivo principal del curso. Desde aquí se ejecutan y organizan los diferentes ejemplos desarrollados a lo largo del aprendizaje de Node.js, permitiendo probar y comprender los conceptos implementados en cada módulo.

## 📂 Carpetas  
- **`assets/`**: Contiene los recursos utilizados durante el curso. Dentro de esta carpeta se encuentra **`docs/`**, donde se almacenan diferentes archivos de texto utilizados para practicar operaciones del sistema de archivos (`File System`) como lectura, escritura, modificación y eliminación de archivos mediante Node.js.

- **`src/`**: Contiene los ejemplos desarrollados durante el curso. Entre ellos se encuentran archivos como **`express.js`**, donde se practican los conceptos relacionados con Express.js, así como **`importaciones y exportaciones.js`**, utilizado para comprender el funcionamiento de los módulos, las importaciones y exportaciones en Node.js mediante CommonJS y ES Modules.
