# 🟢 Node.js Course  

## 📖 Course and Project Description  
This repository is composed of two parts:

1. **Node.js Course**:  
   In the **`course/`** folder, you will find my personal notes on the fundamental and practical concepts of Node.js that I learned in the **SergieCode** course. This course provides the essential knowledge needed to understand backend development with JavaScript, including detailed explanations about the Node.js runtime, npm, package management, CommonJS and ES Modules, file system operations, environment variables, Express.js, middleware, routing, HTTP methods, REST APIs, MongoDB, Mongoose, Docker, deployment, and the integration of TypeScript into modern Node.js applications.

2. **Node.js Projects**:  
   In the **`projects/`** folder, I developed different backend projects inspired by the examples shown in the **SergieCode** course, applying the concepts learned throughout the course to real-world scenarios. These projects include REST API development with Express.js, CRUD operations, MongoDB database integration using Mongoose, middleware implementation, environment variable management, API testing with Postman, deployment, and other backend development practices. Each project was designed to consolidate the acquired knowledge, strengthen backend development skills, and demonstrate practical applications of Node.js in modern server-side environments.
