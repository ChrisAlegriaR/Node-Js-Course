# 🟢 Node.js Course  

## 📖 Course, Examples and Concepts Description  
This repository is composed of one part:

1. **Node.js Course**:  
   In the **`course/`** folder, you will find my personal notes on the fundamental and practical concepts of Node.js that I learned in the **SergieCode** course. This course provides the essential knowledge needed to understand and develop backend applications with JavaScript, including detailed explanations about the Node.js runtime, npm, package management, CommonJS and ES Modules, file system operations, environment variables, Express.js, middleware, routing, HTTP methods, REST APIs, JSON Server, Postman, MongoDB, Mongoose, deployment tools, Docker, and the integration of TypeScript into modern Node.js applications.
